# ResumeAI

ResumeAI is a web-based resume analysis and applicant ranking system developed using HTML, CSS, JavaScript, Python Flask, pdfplumber, and MySQL.

## Features

- Applicant resume analysis
- PDF text extraction
- Skill and keyword matching
- Match score calculation
- Feedback and suggestions
- Applicant analysis history
- Recruiter applicant ranking
- Recruiter job-wise report history
- Login, signup, and profile management

## Technologies Used

- HTML
- CSS
- Bootstrap
- JavaScript
- Python
- Flask
- MySQL
- pdfplumber
- Chart.js
