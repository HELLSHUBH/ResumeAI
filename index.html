HOW IT WORKS

Simple by design

The project follows a clear full-stack workflow from frontend upload to backend analysis.

1
Upload Resume

The applicant uploads a PDF resume through the web interface.

2
Add Job Description

The user enters the job description or recruiter requirements.

3
Analyze Skills

The backend extracts resume text, identifies skills, and compares them with job requirements.

4
View Report

The system returns a match score, matched skills, missing skills, and suggestions.
